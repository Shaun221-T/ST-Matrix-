1. Introduction to Web Development

Web development is the art and science of building websites and web applications that run online. It combines creativity with technical skills to create engaging digital experiences.

Key Areas of Web Development

  • Frontend (Client-side): What users see and interact with (HTML, CSS, JavaScript)
  • Backend (Server-side): The behind-the-scenes logic (databases, servers, APIs)
  • Full-stack Development: Combining both frontend and backend skills

Real-World Analogy

Think of a website as a house: HTML is the structure (walls, roof), CSS is the decoration (paint, furniture), JavaScript is the functionality (electricity, plumbing), and Backend is the foundation and utilities (plumbing system, electrical grid).

Exercise: Website Analysis

Visit 3 websites you use daily (e.g., YouTube, Twitter, Wikipedia). Identify:

  1. Different HTML elements used (headings, paragraphs, images)
  2. CSS styling choices (colors, layout, spacing)
  3. Interactive JavaScript features (buttons, forms, animations)

2. HTML Basics

HTML (HyperText Markup Language) is the skeleton of every webpage. It provides the structure and content for your site.

Basic HTML Structure

Every HTML document follows this fundamental structure:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Webpage</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first webpage.</p>
</body>
</html>

Essential HTML Elements

  • <a href="https://example.com">Link Text</a> - Creates hyperlinks
  • <img src="image.jpg" alt="Description"> - Embeds images
  • <ul><li>Item</li></ul> - Unordered lists
  • <div>Content</div> - Content division container
  • <button>Click Me</button> - Interactive button
  • <input type="text" placeholder="Enter text"> - Text input field

Semantic HTML Tip

Use semantic elements for better accessibility and SEO:

  • <header> for introductory content
  • <nav> for navigation menus
  • <main> for the primary content
  • <article> for self-contained content
  • <footer> for footer information

HTML Practice

Create a simple "About Me" webpage with:

  1. A heading with your name
  2. A short bio paragraph
  3. An image of yourself or an avatar
  4. A list of your hobbies
  5. A link to your favorite website

3. CSS Fundamentals

CSS (Cascading Style Sheets) brings your HTML to life with colors, layouts, and visual effects.

CSS Fundamentals

CSS works by selecting HTML elements and applying styles to them:

/* Select all h1 elements */
h1 {
    color: navy;
    font-size: 2.5rem;
    margin-bottom: 20px;
}

/* Select elements with class="highlight" */
.highlight {
    background-color: yellow;
    padding: 5px 10px;
}

/* Select element with id="main-content" */
#main-content {
    max-width: 800px;
    margin: 0 auto;
}

Key CSS Concepts

  • Box Model: Content, padding, border, margin
  • Flexbox: Modern layout system for one-dimensional layouts
  • Grid: Powerful two-dimensional layout system
  • Responsive Design: Using media queries to adapt to different screen sizes

Responsive Design Tip

Use this media query to apply styles for mobile devices:

@media (max-width: 768px) {
    .container {
        flex-direction: column;
    }
    
    .sidebar {
        width: 100%;
    }
}
